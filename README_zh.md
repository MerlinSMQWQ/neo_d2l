# neo_d2l

这是一个可以在高版本Python上运行的d2l，并且更多支持工程环境而不是Jupter Notebook环境。

主要是因为d2l本身已经很长时间没有维护了，这只是帮助大家更好地学习Deep Learning。

【注意】
- uv并不适合直接管理pytroch以及torch vision，想要使用需要pip安装或者uv pip。
